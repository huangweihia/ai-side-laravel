## 使用方法

### 前台上传头像
1. 访问 `http://localhost:8081/dashboard`
2. 点击头像右下角的相机图标 📷
3. 选择图片文件（支持 jpg、png、gif 格式，最大 2MB）
4. 上传成功后页面刷新，显示新头像

### 后端逻辑
- **控制器：** `app/Http/Controllers/HomeController.php`
- **方法：** `uploadAvatar()`
- **路由：** `POST /profile/upload-avatar`
- **保存位置：** `public/avatars/avatar_{用户 ID}_{时间戳}.{扩展名}`

## 常见问题

### 问题 1：上传失败
**原因：** 目录权限不足  
**解决：** `chmod 777 public/avatars`

### 问题 2：文件太大
**原因：** 图片超过 2MB  
**解决：** 压缩图片后再上传

### 问题 3：格式不支持
**原因：** 上传了非图片文件  
**解决：** 只支持 jpg、png、gif 格式

## 技术实现

### 验证规则
```php
$request->validate([
    'avatar' => 'required|image|mimes:jpeg,png,jpg,gif|max:2048',
]);
```

### 文件保存
```php
$filename = 'avatar_' . $user->id . '_' . time() . '.' . $file->getClientOriginalExtension();
$file->move(public_path('avatars'), $filename);
$user->update(['avatar' => '/avatars/' . $filename]);
```

### 自动删除旧头像
```php
if ($user->avatar && file_exists(public_path($user->avatar))) {
    unlink(public_path($user->avatar));
}
```
