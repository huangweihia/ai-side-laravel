## 🎯 MCP 本地配置（Windows）

在**你的 Windows 电脑**上执行：

```powershell
# 1. 安装 MCPorter
npm install -g mcporter

# 2. 添加 MCP 配置
mcporter config add WebSearch https://dashscope.aliyuncs.com/api/v1/mcps/WebSearch/mcp --transport http --header "Authorization=Bearer sk-sp-1cc99df020544dff875165bba3132df7" --scope home

# 3. 认证（会打开浏览器）
mcporter auth WebSearch

# 4. 确认
mcporter list
```

## 📦 服务器部署注意事项

### 1. 环境变量

在生产服务器 `.env` 中添加：

```env
# MCP 配置
MCP_WEBSEARCH_URL=https://dashscope.aliyuncs.com/api/v1/mcps/WebSearch/mcp
MCP_WEBSEARCH_KEY=sk-sp-1cc99df020544dff875165bba3132df7

# 订阅价格
SUBSCRIPTION_VIP_PRICE=990
SUBSCRIPTION_VIP_CURRENCY=CNY
```

### 2. 定时任务

服务器 crontab 添加（可选）：

```bash
# 每天清理过期搜索日志
0 3 * * * cd /path/to/ai-side-laravel && php artisan schedule:run
```

### 3. 队列配置

如果使用队列处理文档解析：

```env
QUEUE_CONNECTION=redis
```

### 4. 文件存储

确保 storage 目录可写：

```bash
chown -R www-data:www-data storage bootstrap/cache
chmod -R 775 storage bootstrap/cache
```
