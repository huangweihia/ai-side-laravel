## 🔧 部署步骤

### 在 Windows PowerShell 中执行：

```powershell
cd D:\lewan\openclaw-data\workspace\ai-side-laravel

# 1. 运行数据库迁移
docker compose exec php php artisan migrate --force

# 2. 清理缓存
docker compose exec php php artisan config:clear
docker compose exec php php artisan cache:clear
docker compose exec php artisan view:clear
```

## 🎨 新增组件

### 1. VIP 付费墙组件
位置：`resources/views/components/vip-paywall.blade.php`

使用方式：
```blade
<x-vip-paywall :article="$article" :user="auth()->user()" />
```

### 2. 互动栏组件
位置：`resources/views/components/interaction-bar.blade.php`

使用方式：
```blade
<x-interaction-bar :model="$article" :user="auth()->user()" />
```

---

## 📝 文章详情页修改

在 `resources/views/articles/show.blade.php` 中添加：
