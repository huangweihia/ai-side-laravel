## 🔧 部署命令

```powershell
cd D:\lewan\openclaw-data\workspace\ai-side-laravel

# 1. 运行数据库迁移
docker compose exec php php artisan migrate --force

# 2. 清理缓存
docker compose exec php php artisan config:clear
docker compose exec php php artisan cache:clear
docker compose exec php artisan view:clear

# 3. 创建测试用户积分
docker compose exec php php artisan tinker --execute="App\Models\User::all()->each(fn(\$u) => App\Models\UserPoint::firstOrCreate(['user_id' => \$u->id, 'balance' => 500, 'total_earned' => 500]))"
```

## 📊 后台功能位置

### 用户管理
- **路径：** http://localhost:8081/admin/users
- **功能：**
  - 查看用户列表
  - 赠送积分
  - 设置 VIP
  - 查看用户积分

### 积分管理
- **路径：** http://localhost:8081/admin/points-manager
- **功能：**
  - 积分统计
  - 流水记录
  - 筛选查询

### 文章管理
- **路径：** http://localhost:8081/admin/articles
- **功能：**
  - 📥 采集文章按钮（顶部）
  - 标记 VIP 内容

### 项目管理
- **路径：** http://localhost:8081/admin/projects
- **功能：**

- **路径：** http://localhost:8081/admin/email-manager
