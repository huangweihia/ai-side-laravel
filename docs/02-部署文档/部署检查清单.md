# 🚀 部署检查清单

## 执行步骤

### 1️⃣ 清理缓存
```powershell
cd D:\lewan\openclaw-data\workspace\ai-side-laravel

docker compose exec php php artisan view:clear
docker compose exec php php artisan config:clear
docker compose exec php php artisan cache:clear
```

### 2️⃣ 创建头像上传目录
```powershell
docker compose exec php mkdir -p public/avatars
docker compose exec php chmod 777 public/avatars
```

### 3️⃣ 测试功能

#### 前台测试
1. **头像上传**
   - 访问 `http://localhost:8081/dashboard`
   - 点击头像右下角的相机图标
   - 选择图片上传
   - 预期：头像立即更新

2. **个人中心**
   - 访问 `http://localhost:8081/dashboard`
   - 检查是否显示：
     - ✅ 我的收藏（最多 10 条）
     - ✅ 我的评论（最多 10 条）
     - ✅ 浏览历史（最多 10 条）
     - ✅ 统计数据（收藏数、评论数、浏览数）

3. **点赞功能**
   - 访问任意文章详情页
   - 点击"🤍 点赞"按钮
   - 预期：按钮变红，显示"❤️ 已点赞"

4. **收藏功能**
   - 访问任意项目/文章详情页
   - 点击"收藏"按钮
   - 预期：按钮变金色，显示"⭐ 已收藏"

5. **评论功能**
   - 访问任意项目/文章详情页
   - 输入评论内容并发表
   - 预期：页面刷新，评论显示在列表中

#### 后台测试
1. **用户管理**
   - 访问 `http://localhost:8081/admin`
   - 左侧菜单应有"用户管理"
   - 可以查看、编辑、删除用户

2. **分享功能已删除**
   - 访问项目详情页
   - 确认右侧没有"分享项目"模块

## 📝 后台管理功能补充（可选）

如需添加评论管理和收藏统计到后台，执行：

```powershell
# 创建评论资源
docker compose exec php php artisan make:filament-resource Comment --generate

# 创建收藏资源
docker compose exec php php artisan make:filament-resource Favorite --generate
```

然后参考 `后台管理功能说明.md` 完善配置。
