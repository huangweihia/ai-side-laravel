## 1. 环境准备

### 1.1 检查 Docker 状态

```powershell
# 检查 Docker 是否运行
docker ps

# 如果没有运行，启动 Docker Desktop
```

### 1.2 启动所有容器

```powershell
cd D:\lewan\openclaw-data\workspace\ai-side-laravel

# 启动所有容器
docker compose up -d

# 查看容器状态
docker compose ps
```

**预期输出：**
```
NAME                STATUS              PORTS
ai-side-php         Up                  0.0.0.0:8080->80/tcp
ai-side-mysql       Up                  0.0.0.0:3306->3306/tcp
ai-side-redis       Up                  0.0.0.0:6379->6379/tcp
```

### 1.3 等待 MySQL 就绪

```powershell
# 等待 10 秒，让 MySQL 完全启动
Start-Sleep -Seconds 10

# 测试 MySQL 连接
docker compose exec mysql mysqladmin ping -h localhost -u ai_side -paiside123456
```

**预期输出：**
```
mysqld is alive
```

## 2. 数据库初始化

### 2.1 检查数据库是否存在

```powershell
# 进入 MySQL 容器
docker compose exec mysql bash

# 登录 MySQL
mysql -u ai_side -paiside123456

# 查看数据库
SHOW DATABASES;
```

**如果看不到 `ai_side` 数据库，执行：**

```sql
CREATE DATABASE ai_side CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
GRANT ALL PRIVILEGES ON ai_side.* TO 'ai_side'@'%' IDENTIFIED BY 'aiside123456';
FLUSH PRIVILEGES;
EXIT;
```

### 2.2 运行数据库迁移

```powershell
# 运行所有迁移（创建表结构）
docker compose exec php php artisan migrate

# 查看迁移状态
docker compose exec php php artisan migrate:status
```

**预期输出：**
```
 INFO  All migrations have been run successfully.
```

### 2.3 运行数据填充（Seeder）

```powershell
# 运行所有 Seeder（填充基础数据）
docker compose exec php php artisan db:seed

# 或者只运行特定的 Seeder
docker compose exec php php artisan db:seed --class=CategorySeeder
docker compose exec php php artisan db:seed --class=UserSeeder
```

**预期输出：**
```
 INFO  Database seeding completed successfully.
```

### 2.4 创建管理员账号

```powershell
# 创建 Filament 管理员
docker compose exec php php artisan make:filament-user

# 按提示输入：
# Name: 海哥
# Email: 2801359160@qq.com
# Password: 设置一个密码（至少 8 位）
```

## 5. 问题记录

### 5.1 数据库连接问题

**错误信息：**
```
SQLSTATE[HY000] [2002] php_network_getaddresses: getaddrinfo for mysql failed
```

**解决方案：**
```powershell
# 1. 检查 MySQL 容器是否运行
docker compose ps mysql

# 2. 如果未运行，重启容器
docker compose restart mysql

# 3. 等待 10 秒
Start-Sleep -Seconds 10

# 4. 再次测试
docker compose exec php php artisan migrate
```

### 5.2 权限问题

**错误信息：**
```
Permission denied
```

**解决方案：**
```powershell
# 修复存储目录权限
docker compose exec php chown -R www-data:www-data storage bootstrap/cache
docker compose exec php chmod -R 775 storage bootstrap/cache
```

### 5.3 路由不存在

**错误信息：**
```
Route [favorites.index] not defined.
```

**解决方案：**
```powershell
# 清理路由缓存
docker compose exec php php artisan route:clear
docker compose exec php php artisan optimize:clear
```
