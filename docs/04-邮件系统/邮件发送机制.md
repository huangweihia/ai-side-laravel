# 📋 邮件发送机制说明

**更新时间：** 2026-03-24 17:30

---

## 📧 当前邮件发送方式

### 1. 手动发送 ✅
**路径：** 后台 → 邮件管理 → 选择模板发送邮件

**触发方式：** 管理员手动点击
**适用场景：** 临时通知、测试邮件

---

### 2. 注册欢迎邮件 ✅
**触发方式：** 用户注册时自动触发
**代码位置：** `app/Http/Controllers/Auth/RegisterController.php`

**流程：**
```
用户注册 → 创建用户 → 创建订阅记录 → 发送欢迎邮件
```

---

### 3. 定时日报 ⏸️（待配置）
**触发方式：** OpenClaw Cron 定时任务

**配置示例：**
```json
{

  "schedule": { 
    "kind": "cron", 
    "expr": "0 2 * * *", 
    "tz": "Asia/Shanghai" 
  },
```


## 🔧 定时任务配置

### OpenClaw Cron 配置

**文件位置：** OpenClaw Gateway 配置

**添加定时任务：**
```bash
# 1. 打开 OpenClaw 管理界面
# 2. 进入 Cron 管理
# 3. 添加新任务

{
  "name": "每日 AI 资讯日报",
  "schedule": { "kind": "cron", "expr": "0 2 * * *" },
  "payload": { "kind": "systemEvent", "text": "php artisan emails:send-daily-digest" },
  "sessionTarget": "main",
  "enabled": true
}
```

### 系统 Crontab 配置

**编辑 crontab：**
```bash
crontab -e
```

**添加任务：**
```bash
# 每天早上 9:00 发送日报（UTC 时间 1:00）
0 1 * * * cd /path/to/ai-side-laravel && php artisan emails:send-daily-digest

# 每周一 9:00 发送周报
0 1 * * 1 cd /path/to/ai-side-laravel && php artisan emails:send-weekly-digest

# 每天凌晨 2:00 采集职位
0 18 * * * cd /path/to/ai-side-laravel && php artisan jobs:fetch-daily
```
