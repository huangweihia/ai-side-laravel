# 🖼️ 头像上传报错修复

## 问题现象
用户反馈头像上传时报错

## 可能原因

### 1. CSRF Token 缺失
**检查：** `resources/views/layouts/app.blade.php` 的 `<head>` 部分  
**解决：** 确保有以下代码：
```html
<meta name="csrf-token" content="{{ csrf_token() }}">
```

### 2. 上传目录不存在或无权限
**检查：**
```bash
docker compose exec php ls -la public/avatars
```

**解决：**
```bash
docker compose exec php mkdir -p public/avatars
docker compose exec php chmod 777 public/avatars
```

### 3. 视图缓存问题
**解决：**
```bash
docker compose exec php php artisan view:clear
docker compose exec php php artisan cache:clear
docker compose exec php php artisan config:clear
```

### 4. 表单缺少 enctype
**检查：** 头像上传表单必须有 `enctype="multipart/form-data"`

**正确代码：**
```html
<form id="avatar-form" action="{{ route('profile.upload-avatar') }}" 
      method="POST" enctype="multipart/form-data" style="display: none;">
    @csrf
    <input type="file" id="avatar-upload" name="avatar" accept="image/*" onchange="this.form.submit()">
</form>
```

### 5. 路由未定义
**检查：** `routes/web.php` 是否有：
```php
Route::post('profile/upload-avatar', [HomeController::class, 'uploadAvatar'])->name('profile.upload-avatar');
```

### 6. 控制器方法缺失
**检查：** `app/Http/Controllers/HomeController.php` 是否有 `uploadAvatar()` 方法

## 常见错误及解决

### 错误 1: 419 Page Expired
**原因：** CSRF Token 无效  
**解决：** 清理缓存，刷新页面

### 错误 2: 500 Internal Server Error
**原因：** 目录权限或代码错误  
**解决：** 检查 `public/avatars` 权限，查看日志

### 错误 3: 404 Not Found
**原因：** 路由未定义  
**解决：** 检查 `routes/web.php`

### 错误 4: 413 Payload Too Large
**原因：** 文件超过 2MB  
**解决：** 压缩图片后再上传
