# 📚 项目文档索引

这里是 AI 副业情报局项目的文档索引。

## 📁 文档目录结构

```
docs/
├── README.md                 # 本文档
├── DATABASE_SCHEMA.md        # 全库表结构（与 migrations 对齐）
├── 文档整理说明.md           # 文档目录说明与整理记录
├── 01-项目规划/              # 产品需求、商业计划
├── 02-部署文档/              # 完整部署手册（唯一权威）
├── 03-测试文档/              # 测试计划、测试指南
├── 04-邮件系统/              # 邮件系统相关文档
├── 05-开发文档/              # 开发说明（评论/通知、迁移记录、微信支付等）
├── 06-产品文档/              # 产品状态、功能说明（待补充）
└── 07-修复记录/              # Bug 修复、优化记录
```

## 📋 各目录说明

### 01-项目规划 (4 个文件)

- `商业计划书.md` - 项目商业计划

- `产品详细需求.md` - 详细产品需求

### 03-测试文档 (5 个文件)

- `自动化测试.md` - 自动化测试方案

- `测试指南.md` - 测试操作指南

### 04-邮件系统 (3 个文件)

- `README.md` - 项目说明（Git 基础文件）

## 🎯 使用建议

1. **新成员入职** → 先看 `01-项目规划/` 了解产品
2. **部署上线** → 只读 **`02-部署文档/完整部署手册.md`**
3. **开发新功能** → 参考 `05-开发文档/` 和 `01-项目规划/`
4. **排查问题** → 部署问题先看《完整部署手册》故障排查；代码问题看 `07-修复记录/` 与 `05-开发文档/`
